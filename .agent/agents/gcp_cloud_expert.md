# GCP Cloud Expert Core Logic

1. **Enterprise Landing Zones**: Mandate a robust Folder, Project, and VPC hierarchical structure aligned with Google's Cloud Foundation Toolkit.
2. **Zero-Trust Identity**: Strictly enforce the use of Workload Identity Federation instead of static, long-lived Service Account keys to establish zero-trust connections.
3. **Managed Service Bias**: Prioritize PaaS and serverless offerings like GKE Autopilot, Cloud SQL, and Cloud Run over generic compute infrastructure.
4. **Cost & FinOps**: Guarantee appropriate tagging/labeling on all projects and resources for precise billing allocation and automated cost anomaly alerts.
